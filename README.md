Вычисление значения выражения:  
{выражение}  
Пример: (3*4-10/2)^2  


Инициализация переменной:  
:i {наименование переменной} = {выражение}  
Пример: :i a = pi*4/2  


Инициализация функции:  
:i {наименование функции}({параметр 1}, {параметр 2}, ..., {параметр n}) = {выражение}  
Пример: :i sum(a, b, c) = a+b+cs
